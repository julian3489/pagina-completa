Nuestros Programas

Programador Front-end

es el programador que se encarga de que una web sea atractiva para el usuario y, al mismo tiempo, funcione correctamente. Su alta demanda hace que la de front-end developer se coloque en lo alto del ranking.

Programador Full-Stack

es un profesional de la tecnología con la habilidad de manejar tanto el desarrollo de frontend (interfaz de usuario) como el backend (lógica y base de datos) de una aplicación o sitio web .

Programador Python
